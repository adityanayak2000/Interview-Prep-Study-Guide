## Start here (short)

1. Pick a track: [Semiconductor / Systems](Study_Guide/archetypes/Semiconductor_Systems.md) or [Software / Product SDE](Study_Guide/archetypes/Software_Product_SDE.md).  
2. Open a **company file** from the [full index below](#complete-company-index) (or the [status board](Study_Guide/companies/README.md)).  
3. Drill **High-weight topics** linked from that company.  
4. Use [DSA Overview](Study_Guide/topics/DSA/00_DSA_Overview.md) pattern hubs + Boss Sheets.

**Suggested first companies (systems):** [Qualcomm](Study_Guide/companies/tier_S_semiconductor/Qualcomm.md) · [TI](Study_Guide/companies/tier_S_semiconductor/Texas_Instruments.md) · [MediaTek](Study_Guide/companies/tier_S_semiconductor/MediaTek.md)

## Contribute in 5 steps

1. Fork / clone this repo; create a feature branch.  
2. Read [CONTRIBUTING.md](CONTRIBUTING.md) + [companies/_TEMPLATE.md](Study_Guide/companies/_TEMPLATE.md).  
3. Gather links (GFG, Discuss, Medium, Drive PDFs).  
4. Paste into an LLM with [prompts/AI_CONTRIBUTOR_BRIEF.md](prompts/AI_CONTRIBUTOR_BRIEF.md).  
5. Open a **Pull Request** to `main`. Direct push to `main` only for **adityanayak2000**.
