# Feedback — Interview Prep Study Guide

Thanks for helping improve this public study guide.

**Repo:** https://github.com/adityanayak2000/Interview-Prep-Study-Guide  
**Prefer:** [GitHub Discussions](https://github.com/adityanayak2000/Interview-Prep-Study-Guide/discussions) for ideas / Q&A · [Issues](https://github.com/adityanayak2000/Interview-Prep-Study-Guide/issues/new/choose) for actionable fixes.

If Discussions is not visible yet, use an Issue — maintainers enable Discussions under **Settings → Features**.

---

## What to report

| Type | Use | Example |
|---|---|---|
| Typo / unclear section | Issue → Content fix | Wrong definition in `topics/OS.md` |
| Broken link | Issue → Broken link | Dead Drive URL in a Resource Panel |
| New resource | Issue → Resource suggestion | New LeetCode Discuss / PDF / GitHub notes |
| Company intel update | Issue → Content fix | Process changed for Flipkart campus |
| Study / roadmap idea | Discussion | “Add a 2-week Tier S sprint” |

## What **not** to send

- Campus **CTC / LPA** tables, offer letters, or salary screenshots  
- Student names, emails, roll numbers, or other **PII**  
- Full dumps of private OA PDFs that identify individuals  
- Requests to publish maintainer-only private plans  

Compensation material is intentionally **out of this repo**. See [CONTRIBUTING.md](CONTRIBUTING.md).

## How maintainers route your feedback

| Your suggestion | Where it lands |
|---|---|
| Topic depth / FAQ | `Study_Guide/topics/<Topic>.md` (keep Boss Sheet 10·3·5) |
| Extra study link | Topic **Resource Panel** (5–15 links) and/or `Study_Guide/00_INDEX.md` curated row |
| Library PDF / Boss Sheet | Root `Interview_Prep_Resources_Context.md` (+ optional `🎯 Interview Prep Resources.md`) |
| Company process / rounds | `Study_Guide/companies/tier_*/…` + status row in `companies/README.md` |
| Doc changed online first | Living Google Doc sync → library deltas only ([`Study_Guide/_legacy/README.md`](Study_Guide/_legacy/README.md)) |
| STL day-before notes | [`Study_Guide/topics/DSA/STL/`](Study_Guide/topics/DSA/STL/) |
