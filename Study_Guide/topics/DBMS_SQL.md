## Why it matters
Product interviews: joins, indexes, transactions. Some semiconductor panels (TI) include SQL MCQs.

## Concept map
- Relational model; keys (super/candidate/primary/foreign); normalization through 3NF.  
- ACID; isolation levels (awareness).  
- Indexes: B+ tree intuition; when indexes hurt writes.  
- Joins: inner/left/right; nested loop vs hash (high level).  
- Concurrency control: locks vs MVCC (awareness) — see also [Concurrency](Concurrency_Multithreading.md).
