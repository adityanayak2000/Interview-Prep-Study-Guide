## Process Overview
Campus: Online assessment (HirePro / AMCAT variants) → Bosch Day PPT → Technical interview → HR (sometimes skipped if tech feedback strong). Locations often Bangalore / Coimbatore / Pune / Hyderabad. Mobility vs IT track can depend on OA + interview scores.

## Online Assessment (OA)
- **Platform:** HirePro (proctored) or AMCAT depending on year/campus.
- **Duration:** ~100–120 minutes common on HirePro reports.
- **Sections:** Aptitude; technical (C/C++, embedded/COA for some); coding (often 2 questions); C/C++ section. Section navigation often locked.
- **Negative marking:** Reported on some HirePro CS variants (~25%).
- **Difficulty:** Medium; breadth + basics.

## Interview Rounds
- Deep project grill; OOPs four pillars with examples.
- Easy coding; SDLC/testing; C vs C++; structure vs union; MCU vs MPU; timers/counters for ECE crossover.
- HR: family, relocation, why Bosch, 5-year plan.

- Know projects cold; OOPs + SE basics outweigh Hard DSA.

- Coding bar is Easy; don’t underprepare C fundamentals.
