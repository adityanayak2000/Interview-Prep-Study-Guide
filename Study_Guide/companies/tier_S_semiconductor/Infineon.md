## Process Overview
OA (Geek Assess / similar; software vs hardware section choice) → resume shortlist → technical → managerial → HR. Embedded focus: OS + C.

## Online Assessment (OA)
- **Platform:** Geek Assess reported (2023 NIT drives); sectioned timed blocks.
- **Format:** Multiple short sections (~14 Q / 20 min × N); choose hardware or software track.
- **Topics (software):** OS, C, aptitude/electronics mix depending on track.
- **Difficulty:** Medium.

## Interview Rounds
- OS-heavy technical (processes, memory, sync).
- Embedded C / role-specific depth.
- Managerial: embedded systems context, prior round feedback.
- Fast same-day loops common on campus.

## Recurring Patterns & Tips
- Prefer OS + C depth over Hard DSA grinding for Infineon embedded.
- Know which track (SW vs HW) the drive is hiring for.
