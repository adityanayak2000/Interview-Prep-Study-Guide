# IBM
**Role(s):** Software Developer / SDE / SDE-1; Associate System Engineer (ASE); Associate Developer (IBM Consulting / CIC); Application Developer; IBM India Software Labs; IBM CIO campus (intern + FTE)
**Tier:** P
**Status:** done-v3

## Process Overview
IBM India fresher hiring is **drive- and track-dependent** across Bangalore, Hyderabad, Pune, and pan-India CIC centers. Core shape: **OA(s) → technical → managerial (± HR)**; ASE/consulting tracks often add English, cognitive games, GD, or psychometric.

| Path | Entry | Typical flow |
|---|---|---|
| Campus SDE / ASE | TPO | HackerRank OA → (± English/GD) → Tech → Managerial → (± HR) |
| Associate Developer (CIC) | Off-campus / campus | HackerRank → cognitive mini-games → English → PI |
| Software Labs | Campus / apply | HackerRank → psychometric → team-matched tech → managerial |
| Off-campus general | careers.ibm.com | Resume → invited assessments (complete within ~7 days) → interviews |

**Campus vs off-campus:** Campus may include GD; off-campus more often cognitive/English/video. Round count **3–5**. Timeline gaps of days–weeks between stages are common.

**M.E./M.Tech CS:** GFG eligibility includes **B.E./B.Tech/M.Tech/MCA**; CIC Associate Developer posts list ME/M.Tech. Same loop as UG with deeper project grilling. No BITS-specific write-up found.

## Online Assessment (OA)
**Multiple formats exist** — confirm invite email; do not assume one national template.

- **Platform:** **HackerRank** most common; cognitive/English may be separate proprietary tools
- **Duration:** ~45–60 min for 2 coding; GFG summary ~65 min with English; Software Labs 60 min / 2 easy codes
- **Sections (variants):** 2 coding (± English); ASE Medium report: 1 coding / 45 min + English; selective Bangalore drive: virtual 2 coding / 60 min then offline SQL/patterns / 80 min; AD off-campus: coding → cognitive games → English
- **Negative marking:** Coding unknown; English negative marking reported in one Medium ASE drive
- **Topics:** Easy–medium DSA (strings, heaps, arrays, greedy); SQL in selective offline rounds
- **Difficulty:** Predominantly easy–medium

## Interview Rounds
1. **Technical (~45–75 min)** — resume/projects; OOP (Java/C++); DSA live; DBMS/SQL; OS; CN basics; Docker/cloud if on resume
2. **Managerial** — puzzles, project architecture, behavioral, career goals
3. **HR** — motivation, relocation flexibility, formalities
4. **GD** — some ASE campus drives only

## Recurring Patterns & Tips
- Fundamentals and **resume depth** beat contest-hard DSA.
- Prep baseline: **2 easy–medium HackerRank** problems; also English/cognitive for ASE/AD.
- OOP depth (virtuals, SOLID) recurs in SDE reports.
- Psychometric: answer consistently; long waits between stages ≠ rejection.
- Pan-India location flexibility expected.
