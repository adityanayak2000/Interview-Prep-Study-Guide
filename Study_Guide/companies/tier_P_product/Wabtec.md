## Process Overview
Sparse but real India signals: **online test → 1–2 technical rounds → managerial → HR** appears across a 2020 GFG embedded on-campus account and a Bengaluru Glassdoor SWE note. Sample size is too small and dated for `done-v3`; treat OA sectioning below as a **single 2020 data point**, not current policy.

## Online Assessment (OA)
_(Embedded Software on-campus, 2020 GFG — one account)_
- **Platform:** Unknown.
- **Duration:** ~90 minutes, three timed sections.
- **Sections:** Aptitude (~25 Q / 35 min); Embedded/CS fundamentals (~20 Q / 30 min — processors, C/C++ output, digital/analog, OS); Coding (1 problem / ~25 min; e.g. bishop attack positions).
- **Negative marking:** Unknown.
- **Difficulty:** Not rated in-source; company-wide Glassdoor “moderate” aggregates are not OA-specific.

## Interview Rounds
- Embedded technical sample: company familiarity, branch background, deep M.Tech project grill, RTOS scheduling / interrupts, live coding (bit/hex manipulation with casting only).
- Bengaluru SWE/embedded note: logical thinking + coding, strong C/C++, heavy project focus.
- Formal HLD/LLD: not found for India SWE reports.
