# MathWorks (EDG)
**Role(s):** Engineer, Engineering Development Group (EDG); EDG Intern (+ possible FTE conversion); Software Developer / Software Engineer on EDG track; CS Track vs Core Engineering Track (some campus visits)
**Tier:** P
**Status:** done-v3

## Process Overview
MathWorks India new-grad hiring for **EDG** (Bangalore / Hyderabad) is a **fundamentals-first** loop: multi-language OOP/C++, medium DSA, soft skills, and **EDG program literacy**. Campus visits often include PPT + GD; official off-campus “How We Hire” drops GD but keeps a code challenge + interview block.

| Path | Entry | Typical flow |
|---|---|---|
| Campus | TPO | OA (often HackerRank) → PPT → GD (often) → Technical → Managerial → HR |
| Off-campus / direct | MathWorks careers | Application → Code Challenge + recorded video → manager talk → ~4 hr virtual interviews |
| Intern | Campus | EDG intern; some FTE + internship bundles reported |

**Campus vs off-campus:** Campus commonly has **GD** and tests PPT content later; off-campus emphasizes code challenge + manager/HR blocks. OA section maps **vary by year** — confirm with placement cell.

**M.E./M.Tech CS:** Official India listings include **Masters/Ph.D.** graduate EDG. Public write-ups are mostly B.Tech/NIT/IIT; IIT-H postgrad and official Masters eligibility confirm PG is in-scope. No BITS-specific EDG report found — use NIT/IIT patterns.

## Online Assessment (OA)
Formats **vary by year/campus**. Do not assume one fixed template.

- **Platform:** **HackerRank** (multiple campus reports); official off-campus “Code Challenge”
- **Duration:** ~90 min / ~1.5 hr common in campus reports
- **Negative marking:** At least one 2025 Medium report says **none**; others unknown
- **Sections (campus variants — synthesized):** Math/aptitude; CS fundamentals (OS, DSA, DBMS); **C/C++** OOP/pointers/STL; Java and/or JavaScript MCQ; **2 coding problems in different languages**; Python often **bonus**
- **Topics tested:** Medium array/string/greedy coding; deep C/C++ output prediction; multi-correct MCQs in some drives
- **Global/off-campus note:** Some secondary blogs describe math MCQ then **Coding OR MATLAB** choice — treat as **not verified for India campus** unless your invite says so
- **Difficulty:** OA coding mostly medium; interviews can reach medium–hard DSA

## Interview Rounds
1. **OA** — multi-section gate
2. **PPT** — EDG program overview (content reused in GD/HR/Managerial)
3. **Group Discussion** — campus common; EDG understanding or abstract topics; listening > dominance
4. **Technical (~60–90 min)** — resume/projects; declare **2 languages**; OOP deep dive (virtuals, pointers, memory); DSA (LL, trees, graphs, BS); live coding or pseudocode
5. **Managerial** — behavioral/situational; why MathWorks/EDG; Bangalore vs Hyderabad
6. **HR** — strengths/weaknesses; explain EDG in your own words; sometimes merged with Managerial

**Variant:** Some campus reports skip GD and run a longer TR with multiple DSA problems.

## Recurring Patterns & Tips
- **Multi-language fluency** matters — OA often forces two languages for two coding questions.
- **OOP depth beats blind LeetCode grind** — virtual functions, pointers, memory tracing.
- Read the official EDG page; PPT notes are exam material.
- Soft skills (GD + MR + HR) are first-class gates.
- No consistent LLD/HLD campus loop reported for India EDG.
- Explain thought process; prefer clear, explicit code over terse tricks.
