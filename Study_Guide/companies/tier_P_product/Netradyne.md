# Netradyne
**Role(s):** SDE-1; Associate SE – Cloud / Cloud Engineer; AI / Data Scientist / ML Engineer; Embedded Systems Engineer; SDET
**Tier:** P
**Status:** done-v3

## Process Overview
Netradyne (fleet AI / computer vision / edge) hires in **Bengaluru** (and cloud roles also Chennai). Campus loops are **role-family dependent** — cloud/backend, SDE-1, and embedded/CV are not the same funnel.

| Path | Entry | Typical flow |
|---|---|---|
| Campus Cloud / Associate SE (2025) | TPO | Resume filter → OA (on premises) → 1–2 offline technical → HR (often same day) |
| Campus SDE-1 (2022) | TPO | **No OA** in report → Tech 1 (pen-paper DSA) → Tech 2 (projects + puzzle) → HR |
| Embedded | Careers / niche boards | Resume → Linux/C/OS technical → HR |
| Off-campus | Careers / LinkedIn | Screening → technical(s); senior: multi-round virtual + F2F |

**Campus vs off-campus:** Campus cloud is structured OA + fast same-day loops. Off-campus varies by level. ET (2022) noted HackerEarth/Kaggle for sourcing — not a confirmed fresher OA platform.

**M.E./M.Tech CS:** Embedded JDs accept **BE/ME or B.Tech/M.Tech**. Strong IISc M.Tech / research pipelines for CV/ML; no BITS-specific interview write-up. Confirm role family when called.

## Online Assessment (OA)
- **Platform:** Unknown (not named in GFG campus reports)
- **Duration:** Unknown
- **Sections (cloud 2025):** **2 coding + MCQs** (technical CS + aptitude); ~30 MCQs in one report
- **Negative marking:** Unknown
- **Topics:** Coding — arrays, DP (difficulty varies easy → medium–hard by cohort); MCQs — DSA, DBMS, OS, CN, aptitude; cloud adds basic Linux
- **Difficulty:** Cohort-dependent; do not assume one bar
- **Note:** SDE-1 2022 report had **no OA** — OA is not universal

## Interview Rounds
### Cloud / Associate SE
1. Resume filter → OA
2. **Technical 1** — AWS/Linux basics, DSA approach, hashmap-style reasoning
3. **Technical 2** — Java + DBMS depth; DNS; file permissions; DevOps views
4. **HR** — product/competitors awareness, Bangalore relocation, learning under pressure

### SDE-1 (2022)
1. Pen-paper DSA (e.g. N-Queens)
2. Projects + puzzle
3. HR

### Embedded
1. Linux/Android, drivers, C/C++, OS internals
2. HR

## Recurring Patterns & Tips
- Prep to the **JD track** (cloud ≠ SDE ≠ embedded ≠ ML).
- Explain approach clearly; pen-paper or approach-only coding is common.
- Cloud track: Linux CLI, DNS, AWS basics, Java/DBMS.
- Embedded/ME lens: C/C++, multithreading, IPC, OS internals.
- Know company product, leadership, competitors for HR.
- Apply only via official careers — company warns against fee-charging agencies.
