## Process Overview
Sophos (cybersecurity) India hiring is **off-campus–heavy** with **selective campus visits**. Bengaluru / Hyderabad hubs; remote-first common. Not a regular mass recruiter at BITS placement lists — ME candidates should treat **careers + LinkedIn** as the primary path.

| Path | Entry | Typical flow |
|---|---|---|
| Off-campus SWE | Careers / LinkedIn | OA → Tech 1 → Tech 2 → HR |
| Campus (historic / niche) | Joint drives / selective colleges | Written/aptitude+security test → (± GD) → Tech → HR |
| Lateral Java SE-2 | Recruiter | Java fundamentals → coding → manager deep-dive (OA may be skipped) |

**Campus vs off-campus:** Campus may include **GD** and cybersecurity MCQs; off-campus SWE-1 reports show **MCQ + C coding** OA then two technicals. Stack is **team-dependent** (C/networking firewall vs Java/cloud).

**M.E./M.Tech CS:** Intern/SDE postings list Bachelor’s/Master’s CS. No dedicated BITS campus slot documented.

## Online Assessment (OA)
Formats **vary by role/year** — do not invent a single template.

- **Platform:** Unknown across sourced reports
- **Duration:** Unknown
- **Negative marking:** Unknown
- **Off-campus SWE-1 (2021 GFG):** 10 MCQ + **2 coding in C only**; easy difficulty reported
- **Campus 2017 (threat track):** Written MCQ — aptitude (40) + networking/assembly (15) + C output (10)
- **Campus/security 2024:** Aptitude + cybersecurity technical; may include GD
- **GFG recruitment summary (2025):** aptitude/reasoning + DSA coding + personality (role-dependent)

## Interview Rounds
1. **OA / written** — gate (format above)
2. **Technical 1** — easy–medium DSA (strings, primes, LL); CS fundamentals; projects
3. **Technical 2** — more coding; deep C/systems or networking if resume/team warrants
4. **HR** — motivation, company knowledge, scenarios

**Java lateral:** Java 8/streams/multithreading/Spring → medium–hard coding → microservices deep-dive.

## Recurring Patterns & Tips
- Match prep to JD: **C/C++ + CN + Linux** for firewall/endpoint; **Java/Go** for cloud/data teams.
- Security fluency (malware, encryption, IR) helps even for SDE.
- DSA bar for SWE-1 is moderate — not a hard-LeetCode grind in sourced reports.
- Resume drives depth — seminar/security/C projects get grilled.
- Off-campus is the realistic BITS-like ME path unless your college has an active drive.
