## Process Overview
**super.money** (Flipkart Group fintech; Bengaluru) runs a **machine-coding-first** loop. Campus intake is mainly via **Flipkart Early Careers internship → conversion**, not a standalone mass campus OA on GFG. Spelling variants: Super.Money, SuperMoney, supermoney.

| Path | Entry | Typical flow |
|---|---|---|
| Off-campus SDE | LinkedIn / recruiters / InterviewVector | Machine coding (± written OA) → PSDS/DSA → LLD/HLD → HM |
| Campus / early career | Flipkart Flintern → super.money intern | 6-month internship → SDE-1 conversion |
| UI Engineer | Recruiter | Machine coding → UI depth → SD → HM |

**Campus vs off-campus:** No verified standalone super.money campus OA on GFG/Code360. Off-campus is machine-coding heavy; campus JD language includes M.Tech CS but public round detail is sparse.

**M.E./M.Tech CS:** Eligible per campus JD wording; same backend/LLD bar as other CS grads.

## Online Assessment (OA)
- **Campus OA (super.money-specific):** Unknown
- **Staffing-agency SDE (Glassdoor, Sept 2025):** ~10 MCQs (SD + complexity) + 1 medium sliding-window + 2 SQL; then tech + HR
- **Machine coding as first gate:** 1.5–2 hr OOD take-home (Google Form / InterviewVector) — often replaces classic OA
- **Platform:** Unknown for dedicated campus; InterviewVector for some machine-coding rounds
- **Negative marking:** Unknown

## Interview Rounds
1. **Machine coding / LLD (~1.5–2 hr)** — signature: **logging library** (sync/async sinks, levels, thread-safe, buffer); also pub-sub / message broker at senior levels
2. **PSDS / DSA (~1 hr)** — often 2 problems (can include hard / CF C–D); senior: ops/SLA/migration scenarios
3. **System design** — HLD (chess matchmaking, Google Forms, caching) and/or LLD (Calendar, fantasy leaderboard)
4. **Hiring manager** — projects, behavioral, culture

## Recurring Patterns & Tips
- Treat **logging library** as the #1 prep artifact; concurrency gaps reject SDE-2+.
- Explain SOLID/patterns after coding — scored, not just correctness.
- Junior loops may accept partial async demos if design is solid.
- Target Flipkart Early Careers for campus ME/UG entry.
- GFG/InterviewBit/TheJobOverflow have no super.money IE — use InterviewExperiences / Glassdoor / CodingKaro.
