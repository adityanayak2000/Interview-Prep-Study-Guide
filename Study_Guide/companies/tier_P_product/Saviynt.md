**Tier:** P
**Status:** done-v3

## Process Overview
Saviynt (IGA / IAM security platform) hires in **Bengaluru** (hybrid). Campus signal is strongest for **interns and Java Developer campus placement**; ME/M.Tech candidates more often see **off-campus / early-career** tracks. GFG/LeetCode Discuss have **no Saviynt IE pages** — process from AmbitionBox + careers aggregators.

| Path | Entry | Typical flow |
|---|---|---|
| Campus Java Dev (Oct 2024) | Placement | Coding test → Technical → HR (~3 rounds, ≤2 weeks) |
| Campus intern (2026 outreach) | TPO / LinkedIn | College engagement (BITS Goa named in outreach); selection steps sparsely documented |
| Off-campus SWE / Solutions | Careers / Lever / LinkedIn | Resume screen → technical assessment → 2–4 interviews |
| Experienced Java | Referral / apply | Assignment → 2× technical → HR |

**Campus vs off-campus:** Campus Java report is a clean **3-round** loop. Off-campus Solutions Engineering emphasizes **security/networking basics + Java/SQL**; product SWE is Java/microservices. Timelines vary (smooth ≤2 weeks vs slow multi-round reports).

**M.E./M.Tech CS:** Several off-campus listings accept **M.Sc/MCA/Masters** alongside B.E/B.Tech. No separate ME campus SDE rubric — treat as off-campus unless placement cell has a slot.

## Interview Rounds
### Campus — Java Developer
1. **Coding test** — 2 medium problems (JS + Java)
2. **Technical** — DSA (e.g. LIS), DB deadlocks, Linux/DevOps commands, sorting complexity
3. **HR** — fit / communication

### Experienced — Senior Java
1. Assignment → Tech (strings, HashMap design) → Tech (scenario design, scaling, DB) → HR

### Solutions / Support tracks
1. Core Java + Spring → IAM concepts + HTTPS certificates → director/behavioral

## Recurring Patterns & Tips
- **Java-first** culture — Spring, SQL, microservices recur.
- Learn **IAM primers** (IGA, RBAC, SSO, AD/LDAP, HTTPS) even for SWE — differentiates Solutions vs product tracks.
- Ask for OA format upfront — public OA platform detail is thin.
- Resume/project honesty matters; campus tip emphasizes clear communication.
- Process scheduling can be inconsistent — keep parallel options until written offer.
