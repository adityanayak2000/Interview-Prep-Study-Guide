# Flipkart (SDE)
**Role(s):** Software Development Engineer 1 (SDE-1); SDE intern pipelines that may convert to FTE
**Tier:** P
**Status:** done-v3

## Process Overview
Flipkart SDE-1 is **DSA-heavy** with optional **machine coding / LLD**. Loop shape varies by channel (campus, GRiD contest, off-campus / Runway-style drives). Treat the invite email as binding.

| Channel | Typical funnel |
|---|---|
| On-campus | Resume gate → OA → 1–2 PSDS → HM/HR |
| GRiD | Profile → coding test → case/build → finals → interviews |
| Off-campus | Resume → OA → 2× PSDS → HM **or** machine coding first → PSDS → HM |

**M.E./M.Tech:** Eligibility commonly includes M.E./M.Tech CSE/IT; few write-ups isolate PG — expect same loop plus deeper project grilling. Timeline OA→offer often **2–6 weeks**.

> Separate track: [Flipkart Data Science](Flipkart_Data_Science.md) (no SDE-style DSA OA as primary gate).

## Online Assessment (OA)
- **Platform (sourced; varies):** HackerRank, SHL, AMCAT, Unstop — check invite
- **Duration:** ~90 minutes most common
- **Format:** 2–3 DSA coding problems (MCQ/aptitude not consistently reported for recent SDE-1)
- **Difficulty:** Medium → Hard
- **Topics:** arrays/strings, greedy, graphs, trees, DP, stacks, hashing, implementation stories
- **Languages:** Java, Python, C/C++ commonly supported

## Interview Rounds
1. **OA** — gate
2. **PSDS (×1–2, 45–60 min)** — live coding (Doc / CoderPad); approach + complexity + edges; resume deep-dive
3. **Machine coding / LLD (~90–120 min)** — present in many SDE-1 loops, not universal; parking lot, expense splitter, booking MVP, snake-and-ladder, etc.; runnable modular OOP
4. **Design / techno-managerial** — simplified LLD/data model (more common post-GRiD)
5. **HM / HR** — motivation, ownership, projects; light tech follow-ups

## Topic Weights
| DSA | OS | C/C++ | DBMS | CN | LLD | HLD | Concurrency | Aptitude |
|---|---|---|---|---|---|---|---|---|
| Very High | Low–Med | Med | Low–Med | Low | Med–High | Low | Low–Med | Low |

## Recurring Patterns & Tips
- Prep two tracks: (A) OA + 2× PSDS + HM; (B) machine coding + PSDS + HM.
- OA: aim for 2 solid solves in 90 minutes.
- Machine coding: ship must-haves first; explain skipped bonuses.
- Resume integrity — every bullet gets grilled.
- Track GRiD + campus + careers portal in parallel.
