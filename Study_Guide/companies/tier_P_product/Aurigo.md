# Aurigo
**Role(s):** Software Engineer 1 / 2; Senior Software Engineer; SDE Intern → FTE; SDET / SDET Intern; Associate Business Analyst (campus — non-SDE)
**Tier:** P
**Status:** done-v3

## Process Overview
Aurigo (capital planning / construction SaaS; Bengaluru & Mysuru R&D) hires from **BITS Goa, BITS Hyderabad, NIT Trichy** and others (press release). Public **SDE campus OA/round maps are thin**; best campus write-up is Associate BA. Off-campus experienced path includes **AuriCodeCrafters** hackathon (DoSelect). Stack is **Microsoft / .NET**-centric.

| Path | Entry | Typical flow |
|---|---|---|
| Campus (SDE) | TPO | Round sequence **Unknown** for SDE; BA campus: aptitude → 2× 1:1 → HR |
| Intern → FTE | Campus / apply | ~6-month intern → SE-1 conversion (profile pattern) |
| Off-campus experienced | careers.aurigo.com / AuriCodeCrafters | 12-hr DoSelect hackathon (3+ YOE filter) → selection; post-hackathon interviews Unknown |
| Direct apply | careers portal | Resume → interviews (details sparsely documented) |

**Campus vs off-campus:** Campus exists at BITS-like schools; SDE-specific process not published. Hackathon is **not** a fresher campus OA substitute.

**M.E./M.Tech CS:** No ME-specific JD/interview report. FTE SE-1 JDs often list B.E/B.Tech + experience — intern/campus more common for freshers.

## Online Assessment (OA)
- **Campus SDE OA:** Platform, duration, sections, negative marking — **Unknown** (no GFG/LeetCode/Medium SDE write-up)
- **Campus BA proxy:** Aptitude test (easy–medium) reported
- **AuriCodeCrafters:** **DoSelect**, 12-hour online challenge; .NET/C#/JS/SQL Server; 3+ YOE eligibility
- Do **not** invent OA formats; do not confuse with **Auriga IT** GFG posts

## Interview Rounds
**SDE (reconstructed from AmbitionBox snippets + JDs — not a full campus transcript):**
1. Fundamentals / coding — easy–medium arrays, OOP, JS/C# style problems
2. Stack depth — ASP.NET, SQL Server, JS; occasional DevOps (Jenkins)
3. Product/company awareness + projects
4. HR / fit

**Campus BA (sourced Dec 2024):** Aptitude → process-flow 1:1 → HR → data/tables 1:1.

**SDET (separate track):** Java/Selenium/API — do not prep as core SDE unless JD says so.

## Recurring Patterns & Tips
- Prep **C# / ASP.NET / JS / SQL Server**, not Java-only LeetCode grind.
- Easy–medium array/OOP questions in AmbitionBox — not Hard CP.
- Know Masterworks / capital-program-management product context.
- Intern performance is a real conversion lever.
- Confirm OA with placement cell — public SDE OA remains Unknown.
