## Process Overview
Round count is **not fixed** (reports span ~3–6 stages), but the recurring campus skeleton is **OA → 2 technical rounds → senior/managerial (± director) → HR**. Older delivery-track campus accounts and off-campus UI referral loops can look shorter. Group Discussion: not found.

_Caveat: medium confidence — OA layouts differ materially by drive; PRPC product trivia is not confirmed in SWE campus interview transcripts._

## Online Assessment (OA)
Two sourced formats (do not merge into one fake paper):
- **2021 campus drive:** ~90 minutes — ~20 MCQs (OOP, JS, Java, I/O, aptitude) + 3 coding submissions (e.g. triangular subsequence, primes, XOR-style).
- **Associate System Architect drive:** ~2h25m — ~88 questions across Programming MCQ, Aptitude/Logical, Coding submission, Psychometric, English.
- **Platform / negative marking:** Unknown (AMCAT claims in summaries were not verified in article text).

## Interview Rounds
- **DSA:** Easy–medium strings/lists/maps; approach/pseudocode sometimes accepted without full IDE coding; Sudoku / stack-queue applications in some ASA batches.
- **Java depth:** Multithreading, HashMap internals, memory, exceptions, `volatile`, inheritance/JVM.
- **SQL/DBMS:** Live queries; DROP vs DELETE vs TRUNCATE; 3NF schema design in managerial rounds.
- **Networking / HTTP:** OSI overview; GET vs PUT vs POST trade-offs.
- **Frontend UI track:** `this`/bind/call/apply, closures, React lifecycle, promises, CSS flexbox live tasks.
- **Projects:** Deep grill in essentially every account; occasional concurrency/cart-style scenario prompts in senior rounds.
- **Puzzles:** Appear in multiple independent reports.
- **PRPC/Pega BPM glossary content:** Not found in first-hand SWE campus write-ups — exclude marketing Q banks.

## Recurring Patterns & Tips
- Expect Java + SQL depth more than Hard contest graphs for many India campus loops.
- Track matters: ASA vs UI vs backend — prep accordingly.
- Do not waste fresher time memorizing PRPC exam dumps unless the JD is explicitly Pega-platform delivery.
