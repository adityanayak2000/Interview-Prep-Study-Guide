# o9 Solutions
**Role(s):** Associate Software Engineer (campus; Bangalore/Hyderabad)  
**Tier:** G  
**Status:** done-v3

## Process Overview
Common path: resume shortlist → **online test → 2 technical interviews**. CGPA cutoffs ~6.9–7.0 appear in some reports (not universally confirmed).

## Online Assessment (OA)
- **Platform / duration / marking:** Unknown.
- **Content mix:** Aptitude + OS/CN/OOP/DBMS MCQs + code-output (multi-language) + 2 coding problems; one batch also cites heavy aptitude DI + SQL items.

## Interview Rounds
- **R1 (~30 min):** Project walkthrough + “what would you improve” follow-ups.
- **R2:** SQL writing (joins, repeated values), HTML vs HTML5, array/string problems (duplicates, palindrome), OOP + sorting.
- SQL is the strongest recurring theme.

## Topic Weights
| DSA | OS | C/C++ | DBMS | CN | LLD | HLD | Concurrency | Aptitude |
|---|---|---|---|---|---|---|---|---|
| Med | Low | Low | High | Low | None | None | None | Med |

## Recurring Patterns & Tips
- Drill SQL joins/query writing hard.
- Prepare concise project improvement answers for R1.
