# Tesco Bengaluru
**Role(s):** Software Engineer / Associate Software Engineer (India technology center)  
**Tier:** G  
**Status:** done-v3

## Process Overview
Tesco’s Bengaluru tech hiring (distinct from UK retail Glassdoor pools in many listings): **aptitude → online coding (± C/DBMS/DS MCQs) → technical interview(s) → HR**. Some accounts describe four rounds including a project/tech-stack sitting.

## Online Assessment (OA)
- **Platform / marking:** Unknown.
- **Format:** Often ~2 coding questions in ~2 hours plus MCQs on C/C++, DBMS, DS.
- **Difficulty:** Unknown as a single rating; DSA emphasis is high in interviews.

## Interview Rounds
- Strong algorithm focus (DP variants, string subsequence, linked-list palindrome one-pass style problems appear in write-ups).
- Technical covers projects, C++, databases, sometimes shell scripting.
- HR standard.

## Topic Weights
| DSA | OS | C/C++ | DBMS | CN | LLD | HLD | Concurrency | Aptitude |
|---|---|---|---|---|---|---|---|---|
| High | Low | Med | Med | None | None | None | None | Med |

## Recurring Patterns & Tips
- Prioritize classic DP + linked lists + strings.
- Keep basic Linux/shell familiarity if the JD touches DevOps-lite stacks.
