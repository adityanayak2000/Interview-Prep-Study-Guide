## Process Overview
Go Digit General Insurance (not unrelated “Digit” products). Typical virtual campus loop: **online aptitude + coding → 1–2 technical interviews (senior + scenario/PM) → offer**, often finished within about a week on MS Teams.

## Online Assessment (OA)
- **Platform:** Wheebox (named in a GFG account).
- **Format:** ~30 aptitude MCQs + 2 easy Java/coding tasks (timestamp duration, leap year, reverse number, etc.).
- **Negative marking:** Unknown.
- **Difficulty:** Easy–medium; aptitude shortlist matters before coding is weighed.

## Interview Rounds
- **OOP depth:** Four pillars, polymorphism, access modifiers, static, stack vs heap — beyond definitional flashcards.
- **DSA basics:** SLL vs DLL, bubble sort, second-largest, missing number, recursion, tree types/BST/height.
- **SQL:** Joins; second-highest salary style queries.
- **Projects:** Resume deep-dive (e.g. mobile stack choices).
