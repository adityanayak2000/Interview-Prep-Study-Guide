# HCL
**Role(s):** Software Engineer / fresher campus tracks (India)  
**Tier:** F  
**Status:** done-v3

## Process Overview
- Online assessment (elimination round)
- Technical interview (~30 min)
- HR interview (~45 min)
- Some pipelines: extra manager round

## Online Assessment (OA)
- Platform: HackerRank/AMCAT/custom (varies by drive, UNKNOWN which for a given pipeline). ~90-120 min. Sections: Quant Aptitude, Logical Reasoning, Verbal, Computer Fundamentals (OS/CN/DBMS/pseudocode) + 1-2 coding Qs. Negative marking unknown. Difficulty: mostly easy-medium, "very easy with basic to moderate DSA."

## Interview Rounds
- OA: aptitude + CS fundamentals (OS/CN/DBMS) + basic coding
- Tech interview: resume/project questions, basic DSA, OOP basics (overloading/overriding, abstraction), SQL, networking basics
- HR: strengths/weaknesses, relocation, fit

## Topic Weights
| DSA | OS | C/C++ | DBMS | CN | LLD | HLD | Concurrency | Aptitude |
|---|---|---|---|---|---|---|---|---|
| Low | Med | Low | Med | Med | None | None | None | High |

## Recurring Patterns & Tips
- Prioritize aptitude/verbal practice — real filter, not a formality
- Revise OS, DBMS normalization, and CN basics
- Practice easy-medium array/string problems, not hard DSA
- Prepare 2-3 resume projects in depth
