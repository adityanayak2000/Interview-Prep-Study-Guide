## Process Overview
- Eligibility screen (branch/CGPA cutoffs, e.g. 6+ commonly cited)
- Online Assessment (aptitude + coding)
- Technical interview (1, sometimes 2 rounds)
- HR interview
- Some drives: additional GD or business-interpretation round

## Online Assessment (OA)
- Platform: SHL/AMCAT. Duration varies ~70 min (coding-only) to ~3 hrs (full proctored). Sections: Verbal Ability (~12 Q/15 min), Business/Logical Reasoning w/ graphs+charts (~15 Q/25 min), Coding (2 Qs/~1 hr, array/greedy). Negative marking unknown. Difficulty: verbal/logical easy-medium, coding medium.

## Interview Rounds
- DSA: array manipulation, greedy algorithms (medium)
- CS fundamentals: RDBMS vs NoSQL trade-offs, DB system design basics, Java (Servlets, Bean classes, static classes, garbage collection), unit testing, API/integration
- General awareness: tech news, notable startups (soft/behavioral-adjacent)
- HR: motivation, culture fit

## Recurring Patterns & Tips
- Don't neglect aptitude/verbal-reasoning OA section — real filter via SHL/AMCAT gate
- Study DBMS deeply (RDBMS vs NoSQL, normalization, indexing) beyond rote definitions
- Solid Java fundamentals (OOP, garbage collection, servlets) for Java-heavy teams
- Practice easy-medium array/greedy problems rather than hard DP/graph
