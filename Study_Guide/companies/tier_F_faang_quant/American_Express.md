## Process Overview
- Online Assessment — coding + debugging
- Technical round 1 — projects, DBMS, OOP, coding
- Technical round 2 — LLD/DB design + HR-blend or separate HR round
- Some tracks: HireVue behavioral/MCQ round precedes technical rounds
- ~1-1.5 months total

## Online Assessment (OA)
- Platform unknown (varies by drive). ~90 min, 3 questions (typically 1 easy + 2 medium/hard, sometimes a debugging task). Negative marking unknown. Topics: strings, BSTs, arrays, maps/graphs/sets.

## Interview Rounds
- Technical 1: self-intro, project deep-dive (DB-heavy), ACID properties, SQL vs NoSQL, OOP (runtime vs compile-time polymorphism), basic coding
- Technical 2: LeetCode-medium coding, low-level design (DB design), some cybersecurity/Kafka questions tied to project
- HR/behavioral: strengths/weaknesses, goals, situational
- Some drives: HireVue MCQ + recorded behavioral as first filter

## Recurring Patterns & Tips
- Prioritize DBMS fundamentals (ACID, SQL vs NoSQL, query writing) — recurring, high-frequency
- Know your project internals cold — DB choices, tech-stack reasoning heavily probed
- Practice OOP polymorphism (compile-time vs runtime) explicitly
- Be ready for a HireVue-style recorded/MCQ screen on some tracks
