## Process Overview
- Resume/recruiter screen (referral helps, not required)
- Online Assessment for many off-campus/new-grad applicants
- Technical phone screen (1 round, video)
- Virtual onsite: 2 coding rounds + 1 system design + 1 behavioral ("Jedi"/values)
- Some recent loops include AI-assisted coding round (Coderpad AI-assist)
- Hiring committee review → offer

## Online Assessment (OA)
- Platform unknown. ~1 hour, ~4 medium coding questions (arrays/trees/sliding window/BST range sum) reported by one candidate — anecdotal, not universal. Negative marking unknown.

## Interview Rounds
- Phone screen: 2 coding problems/45 min (arrays, trees, sliding window)
- Coding rounds: 2 problems/round, medium-hard DSA
- System design: product-style (feed ranking, notifications), API/schema reasoning
- Behavioral: 5-6 questions + deep follow-ups on 2-3, values-aligned
- Possible AI-coding round evaluating collaboration with AI tools

## Recurring Patterns & Tips
- Practice 2-problems-in-45-min pacing under time pressure
- Prep product-style system design (feed/notifications), not pure infra
- Prep behavioral stories with deep follow-up potential, values-aligned
- Referral helps but off-campus without one is documented as workable
