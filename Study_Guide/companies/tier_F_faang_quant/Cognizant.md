## Process Overview
- Communication round (English fluency/grammar, MCQ + speaking)
- Aptitude round (~30 Qs) + gamified assessment round
- Technical/coding assessment
- Combined Technical + HR interview

## Online Assessment (OA)
- Platform unknown (proprietary/gamified assessment + "Superset" interview platform mentioned). Format: communication test, aptitude MCQs, gamified rounds, technical assessment (~5 Qs: 2 SQL, 2 coding any language, 1 web/frontend for some roles). Negative marking unknown. Difficulty easy to easy-medium.

## Interview Rounds
- Communication: audio repeat-after, grammar/reading MCQs
- Aptitude + gamified: logical reasoning, verbal, quant, cognitive game-based assessment
- Technical assessment: SQL queries, basic coding, sometimes frontend basics
- Interview: OOP (abstract class vs interface, encapsulation/inheritance/polymorphism), C vs C++, DBMS normalization, SQL, project/resume discussion

## Recurring Patterns & Tips
- Communication/English round is a real gate — practice spoken clarity and grammar
- Don't skip the gamified assessment; reach as high a "level" as possible
- Know SQL well — queries expected in technical assessment
- Be ready to explain OOP concepts with real, not textbook, examples
